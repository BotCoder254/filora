# Filora - Product Overview

## Project Purpose
Filora is a full-stack web application combining React frontend with Django backend, designed as a modern web platform with responsive UI capabilities.

## Key Features
- **Modern File Storage**: Secure file upload, organization, and management system
- **User Authentication**: Complete signup/login system with JWT tokens
- **File Management**: Upload, organize files in folders, preview images/documents
- **Sharing & Collaboration**: Generate share links with permission controls
- **REST API Integration**: Full programmatic access via REST endpoints
- **Dark Mode UI**: Twitter-inspired solid dark layers with custom color palette
- **Responsive Design**: Mobile-first design that works on all screen sizes
- **Real-time Updates**: Live file upload progress and status tracking
- **Advanced Search**: Search through files and folders with filtering
- **Storage Analytics**: Visual storage usage tracking and analytics

## Target Users
- Web developers building full-stack applications
- Teams requiring React + Django architecture
- Projects needing rapid prototyping with modern web technologies

## Use Cases
- Web application development with separated frontend/backend
- API-driven applications with React user interfaces
- Projects requiring scalable Django backend with modern React frontend
- Development teams using component-based architecture patterns
