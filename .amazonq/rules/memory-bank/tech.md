# Filora - Technology Stack

## Programming Languages
- **JavaScript**: Frontend development with React
- **Python**: Backend development with Django framework

## Frontend Technologies
- **React**: 19.2.0 - Modern JavaScript library for UI components
- **React DOM**: 19.2.0 - React rendering for web browsers
- **Create React App**: 5.0.1 - Build toolchain and development server
- **Tailwind CSS**: 3.4.18 - Utility-first CSS framework

## Backend Technologies
- **Django**: Python web framework for backend API
- **ASGI/WSGI**: Application server interfaces for deployment

## Testing Framework
- **Jest**: JavaScript testing framework (via React Scripts)
- **React Testing Library**: 16.3.0 - React component testing utilities
- **Testing Library DOM**: 10.4.1 - DOM testing utilities
- **User Event**: 13.5.0 - User interaction simulation

## Development Tools
- **Web Vitals**: 2.1.4 - Performance monitoring
- **ESLint**: Code linting (configured via React App)
- **Browserslist**: Browser compatibility configuration

## Build System
### Frontend Commands
- `npm start` - Development server (localhost:3000)
- `npm build` - Production build
- `npm test` - Test runner
- `npm eject` - Eject from Create React App

### Backend Commands
- `python manage.py runserver` - Django development server
- `python manage.py migrate` - Database migrations
- `python manage.py collectstatic` - Static file collection

## Browser Support
- **Production**: >0.2%, not dead, not op_mini all
- **Development**: Latest Chrome, Firefox, Safari versions
